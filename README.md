# 窗口透明化工具

| 快捷键 | 功能 |
| :-: | :-: |
| Alt+Left/Right    | 降低/增加透明度 |
| Alt+H | 取消透明度 |
| Alt+S | 设置透明度增量 |
| Ctrl+Alt+Left/Right/Up/Down   | 移动窗口 |
| Ctrl+Alt+S    | 设置窗口移动步长 |
| Shift+Alt+Left/Right/Up/Down  | 缩放窗口 |
| Shift+Alt+S   | 设置窗口缩放步长 |
| Alt+T | 置顶/取消置顶窗口 |
| Alt+E | 显示帮助 |
| Alt+ESC   | 退出 |
